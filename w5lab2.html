A question I have about lab: What other HTML tags will we learn this semester?

Terms

  1. HTML Tag (<html></html>)
    2. Tells user's browser the page loading is an HTML document and is the containing tag for the webpage
  2. Body Tag (<body></body>)
    3. Shows where the viewable part of the HTML webpage is
  3. Strong Tag (<strong></strong>)
    4. Bold
  4. Head Tag (<head></head>)
    5. Tag that holds certain pieces of information that are relevant to the webpage (i.e. page title or link to CSS document)
  5. Title Tag (<title></title>)
    6. What is displayed up at the top of a browser window for a webpage
  6. spinweb
    7. Creates /www directory on Mercury account
  7. spinweb -r
    8. Makes all HTML files viewable
  8. Paragraph Tag (<p></p>)
    9. Content paragraph
  9. Header Tag (<h1></h1>,<h2></h2>,<h3></h3>,<h4></h4>,<h5></h5>,<h6></h6>)
    10. Section titles (different sizes; h1 is the largest, all the way down to h6)
  10. Hyperlink Tag (<a href=""></a>)
    11. Link to a webpage
  11. Image Tag (<img src="" />)
    12. Displays an image
  12. List Tags, Ordered and Unordered (<ol></ol>,<ul></ul>)
    13. The main tag for a list; ordered lists have numbers, while unordered lists use bullet points
  13. List Item Tag (<li></li>)
    14. Tag for individual elements within a list
  14. Line Break Tag (<br />)
    15. Makes a break in the content on a webpage
  15. Horizontal Line Tag (<hr />)
    16. Similar to a line break, yet this tag creates a visible line across the page
  16. Table Tag (<table></table>)
    17. Container tag for a table
  17. Table Column Tag (<td></td>)
    18. Tag used for individual columns within a table
  18. Table Header Tag (<th></th>)
    19. Corresponds to a header inside a table
  19. Table Row Tag (<tr></tr>)
    20. Specifies individual rows within a table
  20. HTML
    21. Webpage content and structure of webpage
  21. CSS
    22. Formatting of webpage and its contained content; Cascading Style Sheets
  22. CSS Selectors (or style link)
    23. Refer to the CSS formatting holders; may correspond to certain classes or IDs, for example
  23. CSS Style Rules
    24. The CSS formatting applied to the HTML document and its content
  24. Class and ID
    25. Refer to groups of or individual elements that are to be assigned certain CSS rules (i.e. .selector { CSS rules })
  25. a:link, a:visited, a:hover, a:active
    26. pseudo-class that determines the CSS set for a link; specifically, used for a link when a link has not yet been visited, has been visited, when hovered upon, and a link that's been clicked by user
  26. Div Tag (<div></div>)
    27. Another content container
  27. Static Positioning
    28. Normal page positioning
  28. Relative Positioning
    29. Positioning relative to webpage dimensions and browser window size; "left" and "top" are used
  29. Absolute Positioning
    30. "left" and "right" w/ "top" and "bottom"; tells where content goes exactly on a webpage
  30. z-index property
    31. Determins the order that elements will be displayed on the screen, regarding the stacking of the HTML elements
  31. CSS Box Model
    32. Margin->border->padding->content
  32. Padding
    33. The white space inside an element; between border and content
  33. Margins
    34. Like padding, but margins refer to the white space OUTSIDE content
  34. Background
    35. Content can have colors or images set for a background
  35. Span Tag (<span></span>)
    36. Inline tag for content styling
  36. Programming
    37. A tool to solve problems
  37. Programming - Process
    38. How something is done
  38. Programming - Progam
    39. A language that allows an occurance of something
  39. Programming - Algorithm
    40. A set of rules for a computer to follow
  40. Programming - To call
    41. Pulling up something stored in computer "memory"
  41. Programming - Command
    42. Do something, such as printing
  42. Programming - Function
    43. Code lines that do something, making something occur
  43. Programming - Strings
    44. Letter/number combinations surrounded by quotations
  44. Programming - Variable
    45. A name that can be used to store some sort of information/data to be used later on in a program
  45. Programming - Fundamentals - Naming Conventions (i.e. naming and spelling)
    46. Names are crucial, and spelling matters.
  46. JES Application Program Area
    47. Where code and functions are written
  47. JES Application Interactions Area
    48. Where you can call functions that have been defined in the JES application program area
  48. JES Application File Menu
    49. Allows one to manage one's files of code; open a saved file; save current file, create new file, load current code of program, and print code of current program
  49. JES Application Edit Menu
    50. Standard editing controls expected in any application
  50. JES Application Watcher Menu
    51. Helpful for debugging and allows the watching of variable values during execution of current program
  51. JES Application MediaTools Menu
    52. Tools that can help one look at various types of media
  52. JES Functions Menu
    53. Holds JES function directory; if user selected function from here, it will be automatically put in the program area and displays help area information for the function
  53. JES Application Window Layout Menu
    54. Lets user change window view of application in view; program, interactions, help, watcher
  54. JES Application Help Menu
    55. Offers various tutorials and topics that can be shown at the help area
  55. fileName = pickAFile() (function)
    56. Let's us pick a file via file system on computer
  56. myPicture = makePicture(fileName) (function) - can substitute pickAFile() in for fileName to use fewer lines
    57. Makes a (technically just opens a specific) picture with the file name of fileName;
  57. show(myPicture) (function)
    58. Displays image
  58. newVariableName = makeEmptyPicture(200,100) (function)
    59. Creates an empty picture
  59. pixel = getPixel(newVariableName,100,50) (function)
    60. Creates a pixel on the empty picture we just created
  60. setColor(pixel,red) (function)
    61. Makes the pixel we've specified red
  61. show(newVariableName) (function)
    62. Shows new picture
